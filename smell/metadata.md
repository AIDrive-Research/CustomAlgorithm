# metadata.json

```json
{
  "name": "air_alcohol_coffee_recognition",
  "version": "968-20260511",
  "category": "olfaction"
}
```

**Custom Algorithm Requirements：**

- `name`：The English name of the algorithm package. It is recommended to keep it consistent with the algorithm package folder name and ensure it does not duplicate any existing algorithm name inside the device.
- `version`：The version number of the algorithm package. It can be defined based on the device model and date, for example, `968-20260511`。
- `category`：The algorithm category. For olfactory algorithms, this is fixed as `olfaction`。
